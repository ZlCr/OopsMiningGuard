# [OMG]我没想挖这个(Oops Mining Guard)
***
## 功能
1.7.10版本专供。调整了挖掘冷却，在生存模式下也有了类似创造模式的挖掘冷却。防止高挖掘速度下失误，破坏不想破坏的方块。  
可通过配置文件修改挖掘冷却时间，可在按键设置中设置切换按键（默认无对应按键）。

## 依赖Mixin
推荐使用
https://github.com/LegacyModdingMC/UniMixins

## Features
Exclusively for Minecraft 1.7.10. Adjusts the mining cooldown, giving survival mode a mining cooldown similar to creative mode. Prevents accidental destruction of unintended blocks when mining at high speeds.  
You can modify the mining cooldown duration via the configuration file, and set a toggle key in the key bindings (default is none).

## Dependency: Mixin
Recommended:  
https://github.com/LegacyModdingMC/UniMixins
